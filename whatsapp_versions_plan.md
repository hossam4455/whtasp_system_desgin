# V1.0 — MVP (الحد الأدنى القابل للاستخدام)
## المدة: أسبوعان
## الهدف: الموظفون يستطيعون الرد على العملاء من داخل النظام

### الميزات (Features)

| # | الميزة | الوصف | الأولوية |
|---|---|---|---|
| F-1.1 | عرض قائمة المحادثات | كل الأرقام التي راسلت النظام مرتبة حسب آخر رسالة | عالية |
| F-1.2 | البحث في المحادثات | بحث بالاسم أو رقم الهاتف | عالية |
| F-1.3 | عرض سجل المحادثة | كل الرسائل الواردة والصادرة مع رقم معين | عالية |
| F-1.4 | الرد النصي | كتابة رسالة وإرسالها للعميل من داخل النظام | عالية |
| F-1.5 | عرض الصور | الصور الواردة تظهر كـ thumbnail قابل للتكبير | عالية |
| F-1.6 | عرض المستندات | ملفات PDF/DOC تظهر مع زر تحميل | متوسطة |
| F-1.7 | عرض الصوت | رسائل الصوت مع مشغّل مضمّن | متوسطة |
| F-1.8 | فلتر غير مقروء | عرض المحادثات التي فيها رسائل واردة بدون رد | عالية |
| F-1.9 | Badge عدد غير المقروءة | عدد الرسائل الجديدة لكل محادثة | عالية |
| F-1.10 | فاصل التاريخ | فاصل بين الأيام في المحادثة | منخفضة |

# V2.0 — Smart (التوزيع الذكي)
## المدة: أسبوعان
## الهدف: كل رسالة واردة لها مسؤول واضح + تتبع كامل

### الميزات الجديدة

| # | الميزة | الوصف | الأولوية |
|---|---|---|---|
| F-2.1 | تعيين تلقائي | رسالة واردة → تُعيَّن تلقائياً للموظف المناسب | عالية |
| F-2.2 | ربط بالمستأجر/المالك | ربط الرقم بـ Tenant/Owner تلقائياً من WhatsApp Contact | عالية |
| F-2.3 | ربط بالعقار | عرض العقار والوحدة المرتبطة | عالية |
| F-2.4 | فلتر "لي فقط" | الموظف يرى فقط المحادثات المُعيَّنة له | عالية |
| F-2.5 | إعادة تعيين يدوي | المشرف يعيد تعيين محادثة لموظف آخر | عالية |
| F-2.6 | تتبع أول قراءة | تسجيل أول من فتح الرسالة + الوقت | عالية |
| F-2.7 | تتبع الرد | تسجيل من رد + وقت الرد | عالية |
| F-2.8 | وقت الاستجابة | حساب الوقت بين الرسالة الواردة والرد (بالدقائق) | عالية |

| F-2.13 | إنشاء فرق عمل | WhatsApp Team مع أعضاء | عالية |
